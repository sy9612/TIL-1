# PORT

자주 사용하는 Port Number

| Name       | Description                 | TCP PORT |
| ---------- | --------------------------- | -------- |
| FTP - 송신 | 파일 전송                   | 20       |
| FTP - 수신 | 파일 수신                   | 21       |
| SSH        | 보안 Shell                  | 22       |
| SMTP       | 이메일 전송                 | 25       |
| DNS        | 도메인 -> IP 변경           | 53       |
| HTTP       | 웹 페이지 전송              | 80       |
| Windows    | 윈도우 연결 (원격 데스크톱) | 3389     |

 필요한 포트들은 이후 추가 예정
